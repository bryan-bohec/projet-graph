# Projet Graph

Ce projet comprends une interface web et une partie console qui peut être accéder avec npm

voici le lien d'installation si nécessaire :
https://docs.npmjs.com/downloading-and-installing-node-js-and-npm

la ligne de commande suivante est nécessaire pour installer les dépendances du projet
```
npm install
```

lancer la ligne de commande suivante à la racine pour avoir accès à notre interface web comprenant notre carte
```
npm run dev
```

lancer la ligne de commande pour avoir accès à l'éxecution console du projet
```
cd .\src\utils\js\
node .\main.js
```
